# WUI
WUI是基于XMod引擎2.0及以上版本的UI扩展库。

目前支持的扩展UI列表：

* `wui.Button`按钮
* `wui.Radio`单选
* `wui.Checkbox`勾选框
* `wui.CheckboxList`勾选框列表
* `wui.TabBar`标签栏A
* `wui.TabPage`标签栏B
* `wui.GridSelect`多选框
* `wui.DropDown`下拉框（v1.1.0新增）

用法参见test.lua中使用例子。

详细中文文档介绍：[WUI — 官方UI扩展模块](https://www.yuque.com/irvinpang/xxzhushou/xmod_api_ext_wui)
