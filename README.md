# WUI
WUI是基于XMod引擎2.0及以上版本的UI扩展库。

目前支持的扩展UI列表：

* `wui.Button`按钮
* `wui.Radio`单选
* `wui.Checkbox`勾选框
* `wui.CheckboxList`勾选框列表
* `wui.TabBar`标签栏A
* `wui.TabPage`标签栏B
* `wui.GridSelect `多选框

用法参见test.lua中使用例子。
